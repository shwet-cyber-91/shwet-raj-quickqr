# QuickQR — Free QR Code Generator

A free, no-signup QR code generator. Type a link or any text, get a downloadable QR code instantly — no ads, no account, no watermark.

## Live

https://shwet-raj-quickqr.vercel.app

## Features

- Live QR preview as you type
- Customizable size and color
- Download as PNG
- Built with [qrcodejs](https://github.com/davidshimjs/qrcodejs)

## Tech

HTML5 · CSS3 · JavaScript (vanilla) · deployed on [Vercel](https://vercel.com)

## Running locally

No build step needed — just open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```
